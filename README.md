# Talk

Talk is a lightweight Discord-like messaging app for the web with:

- Real-time chat powered by Socket.IO
- Multiple channels (`#general`, `#random`, `#announcements`)
- Live online presence list
- Modern dark UI inspired by team chat apps

## Run locally

```bash
npm install
npm start
```

Then open <http://localhost:3000> in one or more tabs.

## Tech stack

- **Backend:** Node.js + Express + Socket.IO
- **Frontend:** HTML, CSS, vanilla JavaScript
